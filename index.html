Schéma Fonctionnel
d'un Ordinateur

Découvre comment un ordinateur reçoit, traite, renvoie et stocke les informations.

🎯 Objectif : Comprendre que l'ordinateur fonctionne comme une chaîne : il reçoit des données, les traite, produit un résultat et peut le sauvegarder.

⌨️ Entrée

Les périphériques d'entrée permettent de saisir des données dans l'ordinateur. Par exemple, quand tu tapes un texte au clavier, les lettres sont envoyées vers le processeur.

🧠 Traitement (CPU)

Le processeur (CPU) est le cœur de l'ordinateur. Il exécute les calculs et suit les instructions des programmes. Sans lui, rien ne fonctionne !

🖥️ Sortie

Les périphériques de sortie permettent de voir ou entendre le résultat. L'écran affiche l'image, les haut-parleurs jouent le son.

💿 Stockage

Le stockage conserve tes fichiers même quand l'ordinateur est éteint. Le disque dur, le SSD ou la clé USB sont des espaces de mémoire permanente.

🏭 L'analogie de l'usine

Imagine une usine qui fabrique des voitures 🚗 :
Entrée → Les matières premières arrivent (acier, verre, pneus) ;
Traitement → Les machines assemblent et transforment les pièces ;
Sortie → Les voitures finies sortent de l'usine ;
Stockage → Les plans de fabrication sont conservés dans les archives.
